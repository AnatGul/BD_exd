# Bangladesh Export Declaration Processor

Инструмент для OCR-распознавания и перевода экспортных деклараций Бангладеш на русский язык.

## Возможности

- OCR-распознавание сканов экспортных деклараций (JPG, PNG, PDF)
- Извлечение данных по 51 полю
- Перевод значений на русский язык с учётом правил трансформации адресов и названий компаний
- Экспорт в Excel (3 колонки: Поле / Исходное значение / Перевод)

## Требования

- Python 3.12+
- Windows

## Установка

```bash
pip install openpyxl python-docx pdfplumber PyPDF2 easyocr Pillow
```

### Альтернативная установка (без интернета)

Скопируйте папку `site-packages` из системного Python:
```bash
xcopy /s /y "E:\python3121\Lib\site-packages\*" ".venv\Lib\site-packages\"
```

## Использование

### Из командной строки

```bash
# Обработать все файлы в текущей папке
python exdBD.py

# Обработать файлы из указанной папки
python exdBD.py D:\scans

# Обработать один файл
python exdBD.py D:\scans\declaration.jpg

# Указать выходную папку
python exdBD.py D:\scans -o D:\output
```

### Из PyCharm

1. Откройте проект в PyCharm
2. `File → Settings → Project → Python Interpreter`
3. Добавьте System Python: `E:\python3121\python.exe`
4. Установите зависимости ( см. выше)
5. Запустите `exdBD.py`

### Через bat-файл

```bash
run.bat
```

## Правила перевода

### Не переводятся (оставляются в оригинале)

- Номер документа
- Код офиса экспорта
- Код страны получателя
- Условия поставки
- Код условий поставки
- Код перевозчика
- Код банка
- Сектор и фонд
- Номер места
- Номер коносамента
- Код ТН ВЭД
- Код CPC
- Код валюты
- DFIC/VAT
- И другие кодовые поля

### Трансформация адресов

Россия:
```
Stachki avenue 184, 344090 Rostov-on-Don City, Russia
→ Российская Федерация, г. Ростов-на-Дону, пр. Стачки, 184
```

Бангладеш:
```
Kandail, Satgram, Narsingdi Sadar; Madhabdi PS; Narshingdi-1603; Bangladesh
→ Бангладеш, р-н Нарсингди, г. Нарсингди-1603
```

### Трансформация названий компаний

```
JSC "Gloria Jeans Corporation" → АО "КОРПОРАЦИЯ ГЛОРИЯ ДЖИНС"
Asdwa Fashion Ltd. → Асдва Фэшн Лтд.
```

### Перевод стран

```
Bangladesh → Бангладеш
Russia → Россия
```

### Перевод Incoterms

```
FOB → FOB (свободно на борту)
```
